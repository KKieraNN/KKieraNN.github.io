Microsoft Project

MS Project is a project management software that must be installed on a Window PC, it features a detailed tabbed navigation toolbar and it allows a user to view task through a Gantt chart, network diagram, calendar, resource sheet and has features like resource usage.

When a user creates a new project, MS project provides the user with all the required fields like task Description, Start Date, Finish Date etc. It also allows the users to add extra fields like baseline cost to the project.

MS Project allows the users to auto schedule their task if provided with task durations and dependencies, this feature allows the user to plan out their project in detail in the planning phase.

Task can be assigned to team members using the assign resources tab, MS project also allows users to assign members to a task part time. Task can be added or removed easily by right clicking the task file and linked using the link task icon in the taskbar.

MS Project automatically calculates cost for each task and the overall cost of the project when supplied with hourly rates to people/resources and any delays in the executing phase are automatically reflected on the overall cost of the task/project. User can also view the workload assigned to each member and easily makes changes as he sees fit.

User can also generate reports at the end of their project like Project summary, Milestone report, Budget report, Over Budget resources or create a custom report.
